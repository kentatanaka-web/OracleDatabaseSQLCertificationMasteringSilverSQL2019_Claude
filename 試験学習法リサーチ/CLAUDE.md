# 試験学習法リサーチ

## 目的
Oracle Silver SQL合格者・他資格取得者から学習方法・攻略法をリサーチし、効率的な学習戦略を構築する。

## リサーチ対象
- Oracle Silver SQL（1Z0-071）合格者の体験談・勉強法
- 他SQL系資格取得者（MySQL、PostgreSQL等）のアプローチ
- 参考書・問題集・オンライン教材のレビュー・比較
- 学習期間・スケジュールの実例

## リサーチ収集先（例）
- Qiita・Zenn・ブログ記事（合格体験記）
- X（旧Twitter）の合格報告・勉強法ツイート
- Reddit / Stack Overflow
- YouTube（解説・勉強法動画）
- 資格取得コミュニティ・勉強会

## ファイル構成（予定）
```
試験学習法リサーチ/
├── CLAUDE.md               # このファイル

└── 学習戦略メモ.md         # リサーチから導いた自分の戦略
```
