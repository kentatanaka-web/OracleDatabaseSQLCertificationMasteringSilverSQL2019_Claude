# 教材レビュー

リサーチ日：2026-06-23

## 教材一覧と評価

### 1. 黒本（オラクルマスター教科書）★★★★★ 必須

**正式名称**：オラクルマスター教科書 Silver SQL Oracle Database SQL
**出版社**：翔泳社（株式会社コーソル 企画・日本オラクル監修）
**Amazon**：https://www.amazon.co.jp/dp/4798172367

| 項目 | 内容 |
|------|------|
| 言語 | 日本語 |
| 対応試験 | 1Z0-071（最新版対応） |
| 特徴 | 章末確認問題＋模擬試験付き、図解豊富 |
| 用途 | 基礎固め（1〜3周） |

**合格者の評価**：
- 「試験範囲との対応度が高い」
- 「細かい箇所まで出題されるため隅々まで読む必要あり」
- 問題数が少ないため、問題集との併用が必須

---

### 2. Ping-t ★★★★★ 必須（無料）

**URL**：https://mondai.ping-t.com/

| 項目 | 内容 |
|------|------|
| 料金 | 無料（2025年時点） |
| 問題数 | 580問以上 |
| 対応端末 | PC・スマホ両対応 |
| 機能 | 間違い問題のみ出題・お気に入り登録 |

**合格者の評価**：
- 「スキマ時間に最適」（通勤時間での活用が多数）
- 「解説が丁寧でわかりやすい」
- 「正答率7〜8割で受験目安」
- **ほぼ全員が使用している最重要ツール**

---

### 3. Udemy 模擬問題集 ★★★★

**コース名**：1Z0-071: ORACLE MASTER Silver SQL模擬問題集（6回分277問）
**URL**：https://www.udemy.com/course/1z0-071-oracle-master-silver-sql6277/

| 項目 | 内容 |
|------|------|
| 料金 | Udemyセール時（1,500〜2,000円程度） |
| 問題数 | 277問（6回分） |
| 用途 | 直前の仕上げ・総合演習 |

**評価**：黒本＋Ping-tに加えて演習量を増やしたい場合に有効

---

### 4. Oracle University 公式動画 ★★★（補助）

**URL**：Oracle University（オラクル公式サイト）

| 項目 | 内容 |
|------|------|
| 料金 | 一部無料 |
| 用途 | 黒本やPing-tで理解できなかった箇所の補完 |

**評価**：
- 「実際の動作を見ながら学べる」
- 「理解しづらいトピックに特に有効」
- メインの教材ではなく補助として使う

---

### 5. Oracle Live SQL ★★★（実践練習）

**URL**：https://livesql.oracle.com/

| 項目 | 内容 |
|------|------|
| 料金 | 無料 |
| 用途 | ブラウザ上でSQLを実際に動かせる環境 |

**評価**：
- 環境構築不要でSQLを試せる
- スキマ時間にフラッシュカード、まとまった時間に実装練習として推奨

---

## おすすめ教材の優先順位

```
必須：黒本（基礎）+ Ping-t（問題演習）
補助：Oracle University動画（理解補完）
余裕があれば：Udemy問題集（演習量増加）+ Oracle Live SQL（実践）
```
