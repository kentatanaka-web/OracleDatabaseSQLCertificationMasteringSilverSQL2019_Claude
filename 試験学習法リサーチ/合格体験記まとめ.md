## 合格体験記パターン比較

### パターンA：1ヶ月合格（IT経験者）

- **学習期間**：1ヶ月
- **学習時間**：40〜60時間
- **背景**：Java Gold SE 11など他資格経験あり
- **方法**：通勤時間＋休日のPing-t中心
- **ポイント**：既存IT知識をベースに効率学習

### パターンB：3ヶ月合格（未経験者）

- **学習期間**：約3ヶ月
- **学習時間**：80〜100時間
- **方法**：黒本3周 → Ping-t周回
  - 本番1ヶ月前まで：黒本を1日30分
  - 直前1ヶ月：平日1〜2時間、休日1〜3時間
- **感想**：「問題集をしっかり解けば難しくない」

### パターンC：一度失敗→合格（44時間）

- **1回目**：55%で不合格（思ったより難しかった）
- **2回目**：74%で合格（合計44時間学習）
- **教訓**：黒本の細かい箇所まで出題される、隅々まで理解が必要
- **方法**：黒本3周 + Ping-t + Oracle University動画

## 合格者に共通する学習パターン

1. **黒本（参考書）で基礎固め** → 最低2〜3周
2. **Ping-t（無料問題集）で実戦練習** → 正答率7〜8割を目安
3. **間違い問題の解説を必ず確認**（暗記ではなく理解）
4. **スキマ時間活用**（Ping-tはスマホ対応）

---

## 難しかった点（要注意ポイント）

- NULL値の挙動（IS NULL / IS NOT NULL / NVL関数）
- 複数選択問題（全部正解しないと得点にならない）
- TRUNCATEとDELETEの違い
- 副問合せの種類（単一行・複数行・相関）
- 変換関数の書式マスク（TO_CHAR等）
